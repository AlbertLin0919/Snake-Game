# Snake-Game


## 使用 HTML、CSS、JavaScript 製作出一個貪食蛇的遊戲
遊戲中只要碰到自己的身體就會結束遊戲，每吃下一顆果實後，都會增加些許速度。<br>
遊戲中會紀錄當前分數以及使用者當前電腦的最高紀錄。<br>
快來挑戰看看你的最高紀錄吧!
